### 侧边栏菜单响应优化

**问题原因：**
- 使用了 `docsify-sidebar-collapse` 插件，该插件在每次点击时进行大量 DOM 操作和计算，导致响应延迟

**尝试的解决方案：**
1. 曾尝试禁用了 `docsify-sidebar-collapse` 插件，改用纯 CSS + 轻量级 JavaScript 实现
2. 用户要求继续使用原插件，已恢复

**最终配置：**
- 恢复使用 `docsify-sidebar-collapse` 插件
- 添加配置 `sidebarDisplayLevel: 0` 默认折叠所有子菜单
- 移除了自定义的折叠 CSS 和 JS
